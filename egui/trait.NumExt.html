Trait egui::NumExt

source · 
pub trait NumExt {
    fn at_least(self, lower_limit: Self) -> Self;
    fn at_most(self, upper_limit: Self) -> Self;
}
Expand description

Extends f32, Vec2 etc with at_least and at_most as aliases for max and min.
